### Ordered List

## Task Lists

- [x] Task 1
- [x] Task 2
- [ ] Task 3

You can create task lists by using checkboxes with square brackets like `[ ]` for an unchecked item and `[x]` for a checked item. Users can click on the checkboxes to mark tasks as completed.
